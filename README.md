# Upload file to Mac

> a simple uploader

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run dev
```

### Compiles and minifies for production

```bash
npm run build
```

## Usage

```bash
git clone https://github.com/zgia/upload2me
cd upload2me
npm i
npm run dev
```

See `src/styles/element/index.scss`.
